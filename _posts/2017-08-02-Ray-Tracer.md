## Description
A python implementation for a ray Tracer. The foundations for this project can be found on *Fundamentals of Computer Graphics* by Peter Shirley, et. al. and *Ray Tracing from the Ground Up* by Kevin Suffern. This is an ongoing project, with the hopes to create a real time real tracer and a path tracer.

### Libraries
- PyQt5
