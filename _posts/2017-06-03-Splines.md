## Description
A set of user friendly interactive tools to draw splines by implementing various algorithms. This project was inspired by Gerald Farin's *Curves and Surfaces for CAGD*.

### Implemented Splines
- Hermite 
- Catmull Rom
- Bezier
- B-Spline
- Nurbs

### Libraries
- PyOpenGL
- virtualenv
- PyQt5
